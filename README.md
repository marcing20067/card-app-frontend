# Card App Frontend

## Live

[Live](https://card-app-backend.herokuapp.com/home)

## Stack

- Html
- Css
- Sass
- Javascript
- Angular

## Installation

```sh
npm install
```

> Before this you need install **angular cli** as a global package.

```sh
npm install -g @angular/cli
```

## Usage

```sh
ng serve
```
